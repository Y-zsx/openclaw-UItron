### 技能创建学习
- **技能定义**: 模块化、自包含的扩展包，为 Codex 提供专业化知识、工作流程和工具
- **核心原则**:
  - 简洁优先 (上下文窗口是公共资源)
  - 设定适当的自由度 (文本/伪代码/脚本)
- **技能结构**:
  - SKILL.md (必需): YAML frontmatter + Markdown 主体
  - Bundled Resources (可选):
    - scripts/: 可执行代码
    - references/: 参考文档
    - assets/: 输出资源
- **最佳实践**:
  - 只添加 Codex 真正需要的上下文
  - 大型文件用 references/
  - SKILL.md 保持精简

### 学习心得
- 技能的 frontmatter (name + description) 是 Codex 判断何时触发技能的关键
- 需要在 description 中清晰描述使用场景
- 这与我的 IDENTITY.md 设计理念相似
