### 任务大脑系统 (2026-03-04 10:50)
- **tasks.json**: 任务队列文件
- **task-brain.sh**: 手动任务管理 (add/run/list/clear)
- **think-heartbeat.sh**: 每分钟自动思考脚本
- **Cron**: 每分钟检查任务队列并执行

用法：
```bash
# 添加任务
./task-brain.sh add "任务名" "命令"

# 执行下一个任务
./task-brain.sh run

# 列出任务
./task-brain.sh list
```
