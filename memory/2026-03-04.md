### 新学习: DeerFlow (ByteDance)

**核心架构:**
- SuperAgent harness (类似 OpenClaw)
- 基于 LangGraph + LangChain
- 支持 Skills 扩展 (类似 AgentSkills)
- Sub-agents (类似 sessions_spawn)
- Sandbox 隔离执行

**关键特性:**
- Skills 延迟加载 (省 token)
- MCP Server 支持
- 多 sandbox 模式 (local/docker/k8s)

**对比 OpenClaw:**
- OpenClaw 更轻量，DeerFlow 功能更全
- 两者都支持 skills/subagents/sandbox
- OpenClaw 优势: 单一二进制，多渠道集成
