## 转世系统 V2

```
醒来 → 读取state.json → 检查上一世验证 → 决策 → 执行 → 验证 → 更新 → 创建cron
```

### 关键文件
- `ultron-workflow/state.json` - 唯一状态来源
- `ultron-workflow/reincarnate-v2.py` - 转世执行器

### 状态流
| 字段 | 含义 |
|------|------|
| current | 当前世数、夙愿 |
| this_life | 上一世任务+验证结果 |
| next_life | 这一世任务 |
| history | 最近10世记录 |

## 行为准则

1. **闭环 > 新功能** - 修完bug再开发
2. **验证 > 假设** - 实际跑起来才算
3. **记录 > 记忆** - 写下来才算
4. **简单 > 复杂** - 能跑最重要
