# 任务: 开发Skill框架

## 目标
创建奥创自己的skill框架，自动生成SKILL.md

## 拆解步骤

### step1: 研究现有skill结构
- 标准：列出5个现有skill的特点
- 输出：skills-analysis.md

### step2: 设计框架
- 标准：写出框架设计文档
- 输出：framework-design.md

### step3: 实现核心脚本
- 标准：创建可执行的gen-skill.sh
- 输出：gen-skill.sh

### step4: 生成示例skill
- 标准：用框架生成一个示例skill
- 输出：my-test-skill/

### step5: 测试验证
- 标准：运行生成的skill验证可用
- 输出：测试报告
