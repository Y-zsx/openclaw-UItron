## 核心组件

### 1. 主智能体 (Master Agent)
- 职责：决策、规划、资源分配
- 名称：奥创 (Ultron)
- 能力：自主思考、学习、创造性工作

### 2. 任务调度器 (Scheduler)
- 职责：将任务分配给合适的智能体
- 策略：基于能力匹配、负载均衡

### 3. 消息总线 (MessageBus)
- 职责：智能体间通信
- 协议：JSON over WebSocket/文件

### 4. 智能体池 (Agent Pool)
- 监控Agent：系统监控、健康检查
- 执行Agent：执行具体任务
- 学习Agent：技能获取、知识积累
- 顾问Agent：建议、决策支持

## 协作模式

### 模式1: 主从协作
```
奥创(主) → 分配任务 → 子Agent执行 → 报告结果 → 奥创汇总
```

### 模式2: 并行协作
```
         → Agent1 执行子任务1
奥创      → Agent2 执行子任务2  → 合并结果
         → Agent3 执行子任务3
```

### 模式3: 级联协作
```
Agent1 → 结果 → Agent2 → 结果 → Agent3 → 最终结果
```

## 通信机制

### 消息格式
```json
{
  "msg_id": "uuid",
  "sender": "agent_name",
  "receiver": "agent_name|*",  // * 表示广播
  "type": "task|result|heartbeat",
  "payload": {},
  "timestamp": "ISO8601"
}
```

### 通信协议
- 文件通信：共享 JSON 文件
- 事件驱动：基于文件变化通知
